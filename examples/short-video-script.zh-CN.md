# 短视频展示脚本

## 标题

我做了一个每周提醒我“人生还剩多少格”的 AI Skill

## 30 秒版本

```text
我做了一个 OpenClaw Skill。

它不会帮你提高效率。
它只会在每个星期天问你一次：

你的人生又少了一格。
这一格，要不要留下些什么？

输入生日之后，
它会告诉你已经活了多少天，
已经用掉多少周，
还剩大约多少周。

这不是死亡倒计时。
这是提醒：

有些星期，不该空白地过去。
```

## 录屏节奏

```text
0-3 秒：打开 README 的分享卡片。
3-8 秒：运行 setup 和 read，展示生命周历数字。
8-15 秒：运行 checkin，展示 sharp 语气。
15-22 秒：输入一周一句话，保存 journal。
22-28 秒：运行 share，展示可截图卡片。
28-30 秒：停在一句话：这一周，有什么值得留下？
```
