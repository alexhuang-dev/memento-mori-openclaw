# Release Notes v0.2.1

## Repository Polish

- Added English and Chinese README files following the structure used in adjacent published projects.
- Added Apache-2.0 license.
- Added security and privacy policy.
- Added `.gitignore` and `.gitattributes` for clean cross-platform publishing.
- Added minimal regression tests and GitHub Actions CI.

## Skill Capabilities

- Life-in-weeks overview and progress bar.
- Weekly check-in command for OpenClaw cron.
- Raw + summary journal storage.
- Recent-week stats and annual review data.
- Markdown/JSON export with optional file output.
- Milestone de-duplication.
- Safety boundary for self-harm or severe hopelessness contexts.

## Privacy Notes

- No sample personal state is committed.
- The script does not make network requests.
- Runtime state and exported journals are ignored by git.
